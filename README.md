# 🚀 Next.js Admin Dashboard

A fully responsive and modern admin dashboard built with **Next.js App Router**, designed for managing data efficiently with interactive visualizations, smooth animations, and scalable architecture.

🔗 **Live Demo:** https://next-admin-dashboard-demo.vercel.app

---

## 📖 Overview

This project demonstrates how to build a production-ready admin panel using modern frontend technologies. It includes data management features, real-time filtering, and a clean UI designed for usability and performance.

---

## ✨ Features

* 📊 **Interactive Charts** – Built with Recharts for dynamic data visualization
* 🎨 **Modern UI** – Styled using Tailwind CSS and shadcn/ui components
* ⚡ **Smooth Animations** – Powered by Framer Motion
* 🔍 **Real-time Search & Filtering**
* 👥 **User Management System**
* 🛠 **CRUD Functionality** (Create, Read, Update, Delete)
* 📱 **Fully Responsive Design** (Mobile, Tablet, Desktop)
* 🧠 **Global State Management** with Redux Toolkit

---

## 🛠 Tech Stack

* **Next.js (App Router)**
* **React**
* **Tailwind CSS**
* **Redux Toolkit**
* **Framer Motion**
* **Recharts**

## ⚙️ Installation & Setup

Clone the repository and install dependencies:

```bash
git clone https://github.com/aadedunmola/data-admin-dashboard.git
cd data-admin-dashboard
npm install
npm run dev
```
