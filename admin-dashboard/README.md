# Data Admin Dashboard

A modern admin dashboard built with **Next.js**, **Tailwind CSS**, and **Recharts**.

## Features

- Collapsible sidebar navigation
- Overview analytics dashboard
- Interactive charts and statistics
- Responsive admin layout

## Tech Stack

- Next.js
- React
- TailwindCSS
- Recharts
- Lucide Icons

## Getting Started

Install dependencies:

```bash
npm install
```
